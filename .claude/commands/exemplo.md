# /exemplo

Este é um modelo de slash command. Copie este arquivo e renomeie pra criar seus próprios comandos.

## Como funciona

1. O Claude lê este arquivo quando você digita `/exemplo`
2. Ele segue os passos na ordem
3. Retorna o resultado

## Passos

1. Ler o arquivo `context/` relevante (se houver)
2. [Descreva o que o comando deve fazer]
3. [Formato do output]

## Output esperado

```
[Descreva como deve ser a saída]
```

## Dicas pra criar seus comandos

- Dê um nome claro: `/relatorio-semanal`, `/analise-vendas`, `/prep-reuniao`
- Seja específico nos passos — o Claude segue literalmente
- Indique quais arquivos ele deve ler
- Defina o formato do output (lista, tabela, texto corrido)
